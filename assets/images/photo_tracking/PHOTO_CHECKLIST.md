## When Adding a New Photo:

### Step 1: Prepare the Photo
- [ ] Photo is high quality (at least 1200px wide)
- [ ] Photo is in JPG format
- [ ] File size is reasonable (< 5MB)
- [ ] Photo is properly rotated/cropped

### Step 2: Name and Move
- [ ] Renamed to correct format (food#.jpg, hike#.jpg, box#.jpg)
- [ ] Copied to correct folder in `assets/images/`
- [ ] Original copy saved in backup location

### Step 3: Log the Photo
- [ ] Added row to MASTER_PHOTO_LOG.csv
- [ ] Filled in: category, filename, subject, date, location
- [ ] Wrote description (this becomes the caption)
- [ ] Added relevant tags
- [ ] Set status to "uploaded"

### Step 4: Add Details (Optional)
- [ ] Updated category-specific CSV (cooking/hiking/boxing)
- [ ] Added extra metadata (recipe, trail info, workout details)

### Step 5: Update Website
- [ ] Verified photo appears on website gallery
- [ ] Checked caption displays correctly on hover
- [ ] Tested on mobile view

### Step 6: Maintain
- [ ] Backed up tracking folder
- [ ] Backed up photos folder
- [ ] Updated status to "published"
